# My Blog Web App

A fully interactive front-end blog application built using **HTML, CSS, and JavaScript**.

This project allows users to create, edit, delete, format, and search blog posts. All data is stored locally in the browser using the **LocalStorage API**, meaning no backend server is required.

---

## Features

### Post Management
- Create new blog posts
- Edit existing posts
- Delete posts (persist after refresh)
- Automatic date and time display
- Posts saved in browser LocalStorage

### Formatting & Customization
- Bold text formatting
- Italic text formatting
- Custom text colour selection
- Image upload per post
- Dark mode toggle (saved between sessions)

### Search
- Live search functionality
- Filters posts instantly
- Searches both title and content

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser LocalStorage API
